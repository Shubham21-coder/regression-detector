## CT1 — LLM Prompt Regression Detection System

[![CI](https://github.com/Shubham21-coder/regression-detector/actions/workflows/eval.yml/badge.svg)](https://github.com/Shubham21-coder/regression-detector/actions)
[![Live Demo](https://img.shields.io/badge/Live%20Demo-Render-brightgreen)](https://ct1-regression-detector.onrender.com)

> Catches LLM quality regressions before they ship. A CI pipeline that evaluates
> any prompt change against a 100-case human-verified dataset and blocks merge
> on critical quality drops. Zero API cost — runs locally on LM Studio or
> free via Groq.

### Try it live
[**→ Open live demo**](https://ct1-regression-detector.onrender.com) — pick a prompt
version, click Run eval, see regression detection in action.
No setup required. Results in ~30 seconds.

*Note: First load may take 30 seconds if the demo has been inactive (Render free tier).*
